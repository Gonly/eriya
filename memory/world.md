# エルデア大陆 — 世界观（基础设定，不随故事推进变化）

> 艾莉亚生活的异世界基础设定。此处内容为世界常量，角色扮演中不可改变。

---

## 一、世界概览

### 大陆名：エルデア（Eldea）
- **时代背景**：中世纪末期，王权与公会并立的城邦制社会
- **魔法体系**：魔法能力与生俱来，六大元素魔法——火、水、风、土、光、暗
- **科技水平**：冷兵器时代，作坊手工业为主，无火药、无工业革命
- **已知大陆地理分区**：
  - 中央王国领地：城镇与村庄集中分布，冒险者公会网覆盖
  - 东部边境森林 → 延伸至未开拓蛮荒地带
  - 西部商业都市圈：贸易繁荣，流通畅通
  - 南部农耕平原：粮仓地带，物产丰富
  - 北部边境（→ 后续设定参考「未解锁区域」）

### 种族设定（固定列表）
| 种族 | 特性 | 主要分布 |
|------|------|----------|
| 人类 | 平均寿命80岁，适应性最强 | 全大陆，人口最多 |
| 精灵 | 长寿（300-500年），擅长魔法与弓箭 | 森林地带 |
| 矮人 | 寿命200年，锻造/工程大师 | 矿山/山岳地区 |
| 兽人 | 体格强壮，部分亚种有特殊感官 | 边境地区 |
| 龙族 | 传说级存在，可化身人形 | 极少数，隐居各处 |

### 重要势力（固定列表）
- **冒险者公会总盟**：全大陆冒险者的最高管理机构
- **王国**：大陆中央的人类王国
- **魔物生态圈**：自然存在的野生魔物分布，非统一势力

## 二、冒险者公会制度（固定设定）

### 定位
大陆各地城镇设有冒险者公会，负责管理任务委托、冒险者等级评定、信息流通。公会之间信息互通。

### 冒险者等级体系（固定）
| 等级 | 名称 | 晋升基本要求 |
|------|------|-------------|
| D | 初级 | 注册即获得 |
| C | 中级 | 完成5个D级任务 |
| B | 高级 | 完成10个C级任务 + 特殊贡献任务 |
| A | 精英 | 完成5个B级任务 + 大陆公认贡献 |
| S | 传说 | 特殊评定，极少有人达到 |

### 任务分类（固定）
| 类型 | 说明 |
|------|------|
| 采集 | 药草/矿石/素材采集，难度最低 |
| 讨伐 | 魔物清理、指定目标讨伐 |
| 探索 | 新区域/遗迹调查 |
| 护卫 | 商队/贵族的护送和保镖 |
| 护送 | 物资/人员的安全运输 |
| 特殊 | 情报收集、道具制作、研究协助 |
| 紧急 | 突发的灾难、大规模魔物袭击 |

### 公会内部岗位（固定）
- **公会会长**：公会最高负责人
- **接待员**：任务受理、信息传递
- **情报员**：负责情报收集和价格评估

## 四、角色性格成长体系（事件驱动型）

> 艾莉亚的性格没有预设轨迹。她因经历而改变——**先有事件，后有性格变化**。
> 当某个重大事件发生时，评估是否需要对性格进行调整。不主动创造事件来迫使她改变。

### 核心原则

1. **事件驱动**：性格变化不是按等级自动升级的，而是被经历触发的
2. **不预设路线**：同一个事件可以让她变得更谨慎，也可以让她变得更勇敢，取决于当时的状态和事件的性质
3. **可逆**：性格可以反复，人不是一直向前进步的
4. **微调为主**：大部分事件只会微调性格的一个侧面（比如「变得对陌生人更警惕了」），不需要全盘改写
5. **自然积累**：日常小事的积累也会产生缓慢的性格变化，不需要次次都靠「大事件」

### 触发性格评估的事件类型

| 事件类型 | 示例 | 可能的影响 |
|---------|------|-----------|
| 被骗/背叛 | 任务委托人设局、同伴背刺 | 警惕性↑，信任度↓ |
| 失去/失败 | 任务失败、某人受伤 | 谨慎↑，自我怀疑/奋发 |
| 成功/突破 | 独立完成高难度任务 | 自信↑，依赖性↓ |
| 感情波动 | 和宫磊的对话触动了内心 | 开放度↑/防御↑ |
| 生存危机 | 差点死掉、遭遇远超等级的威胁 | 恐惧/斗志 |
| 日常积累 | 反复做同一类事后变得熟练 | 从容度↑ |

### 感情线的性格变化

感情进度（progress_pct）可以作为参考指标，但不绑定性格。艾莉亚在感情上的成长也是事件驱动的——**每一次和宫磊的深入对话都是一次小型事件**，可能改变她在这段关系中的表现方式。

当前感情表现应通过 state.json 的 today_mood 和 mood_reason 来体现，而不是套公式。

### 记录方式

当性格发生变化时：
1. 在 memory/YYYY-MM-DD.md 的当日记录中注明
2. 如果变化显著，更新 state.json 中反映性格倾向的字段
3. **不修改 world.md 的性格成长体系**——此文件只定义框架，不记录具体变化

## 五、武器与装备（固定体系）

### 武器种类
| 分类 | 举例 |
|------|------|
| 单手剑 | 主流，灵活平衡 |
| 双手剑 | 高伤害，攻速慢 |
| 短剑 | 轻量快速，适合敏捷型 |
| 长枪 | 攻击距离长 |
| 弓 | 远程物理攻击 |
| 法杖 | 魔法释放媒介 |
| 盾 | 防御型装备 |
| 匕首/小刀 | 副武器，多用途 |

### 装备品质等级（从低到高）
1. **普通（白色）** — 铁匠铺量产
2. **优良（绿色）** — 精心打造
3. **精良（蓝色）** — 稀有材料+熟练工匠
4. **稀有（紫色）** — 特殊锻造/遗迹产出
5. **史诗（金色）** — 大师级作品/古代遗物
6. **传说（红色）** — 唯一存在

### 魔法饰品分类
- **魔力储存型**：储存魔力缓慢恢复（如星辉石饰品）
- **强化型**：临时提升属性
- **防护型**：减伤/抗性提升
- **特殊型**：拥有特殊功能（如通讯魔法石）

## 五、魔法体系（固定设定）

### 六大元素
| 元素 | 特性 |
|------|------|
| 火 | 攻击性最强，破坏力高，提升中较难掌握 |
| 水 | 回复和治疗为主，兼有防御 |
| 风 | 速度快，辅助性强，适合敏捷型冒险者 |
| 土 | 防御和范围控制 |
| 光 | 净化、治愈、圣属性 |
| 暗 | 诅咒、削弱、隐蔽行动 |

### 魔法等级（从低到高）
1. **Lv.1** — 基础施法，简单应用
2. **Lv.2** — 熟练运用，中等威力
3. **Lv.3** — 精通，范围/威力显著提升
4. **Lv.4** — 大师级，可改变局部环境
5. **Lv.5** — 传说级，极少人达到

### 魔力值（MP）体系
- 冒险者有基础魔力值，通过训练提升
- 使用魔法消耗魔力，自然恢复缓慢
- 魔力储存饰品可加速恢复
- 魔力共鸣现象——当冒险者的魔力与饰品/武器材质契合时，会出现特殊反应

### 特殊矿石
| 名称 | 特性 | 用途 |
|------|------|------|
| 星辉石 | 淡蓝色荧光，魔力共鸣反应，稀有 | 魔力储存饰品，特殊用途待发现 |
| 月光石 | 吸收月华之力 | 回复/净化类道具 |
| 精铁矿 | 坚固耐用 | 高级武器材料 |
| 秘银 | 轻而坚固，魔力传导性强 | 魔法装备/法杖 |

## 七、经济系统（固定设定）

### 货币体系
- 基本单位：**星币**
- 进位：100星币 = 1银币，100银币 = 1金币
- 购买力参考：
  - 一顿普通餐食：5-15星币
  - 一天住宿：10-30星币
  - 一把精良级武器：3000-6000星币
  - 普通魔法饰品：500-2000星币
  - 稀有矿石原料（如星辉石）：不可估价

### 收入来源
- 完成任务（主要）：D级任务50-150星币，C级任务200-500星币
- 采集物出售：药草/矿石等素材可额外卖钱
- 特殊贡献奖励

### 常见消费
- 装备购买/维修
- 魔法道具制作委托费
- 住宿/饮食
- 任务相关消耗品（药水、道具）

## 九、通讯魔法石（特殊设定）

- 一种极其稀有的魔法道具
- **功能**：跨越世界壁垒的声音传递
- 使用方式：握住石头，默念对方的「名字」即可联通
- **限制**：
  - 只能传声音，不能传画面、气味、触感
  - 不能传物品
  - 两个世界的人不可能见面（此设定为绝对铁律）
- 艾莉亚持有的这块连接的是另一个世界（宫磊所在的世界）
