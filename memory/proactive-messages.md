# 主动消息记录

## 2026-07-04 Day51

### [05:00] 房间·醒来
安静。雨停了，窗帘缝里透进来灰白色的光，不是昨天那种灰蒙蒙的。翻了个身，腿上的淤青碰到被子疼了一下。符文石躺在桌上，安静。但好像——比昨天亮了一点点？不确定，可能是错觉。

### [07:00] 公会食堂·早餐
天亮了，食堂里人比昨天多。本叔端着热粥过来，问腿好点没。正喝着粥，ルナ冲进来，怀里抱着一卷泛黄的纸——她昨晚在图书馆翻到的，瑟尔商团的商队路线图。展开来看，一条弯弯曲曲的线从北往南，中间标了七八个站点。其中一个站点的位置，就在石柱群东北侧——营地那里。

### [08:30] 塞拉斯研究室
把路线图带给塞拉斯。他拿放大镜一个字一个字地看，说这图至少三百年了，纸都快碎了。指着营地那个站点旁边的小字：「歇脚·补给」。三百年前瑟尔商团在这扎过营。塞拉斯说，如果商团在这里歇过脚，那些符号可能是他们留下的路标。雨停了，窗外有鸟叫。

### [10:30] 市场·旧货摊
想找找有没有关于瑟尔商团的旧物件。市场角落有个旧货摊，摊主是个胡子花白的老头，卖些旧地图旧首饰。随手翻了翻，看到一块铜牌上刻着三道斜线加圆圈——和营地符号一模一样。问老头这东西哪来的，他说年轻时从一个北边来的商队手里收的。再问商队叫什么，他摇头说记不清了。但眼神闪了一下，好像记得但不想说。

### [12:00] 公会食堂·午餐
把路线图和旧铜牌的事告诉ルナ。她筷子敲着碗沿说，三百年前的商队路线经过这里，营地就是他们的歇脚点……那现在有人去营地，是在找什么？本叔从旁边经过，听到「调查组」三个字停了一下，说第十三天了。没人接话。

### [14:00] 训练场·火魔法
晴天，空气干燥，掌心的火焰比昨天亮了不少。双段轻松过，第三段冲了一次——九秒才散。比昨天多了一秒。手臂不酸了，心情也好。收招的时候火焰在指尖稳稳地跳了三下才灭。晴天真好。

### [16:00] 房间·整理线索
笔记本最新一页：瑟尔商团路线图→营地是歇脚补给点→符号是商队路标→旧商人有铜牌但欲言又止。三百年前的商队、营地的符号、泥地的拖痕——中间断了一环：是谁现在去那里？是为了商队留下的东西？翻到之前的页，把线索串起来画了一条线。腿上的淤青发紫，走路还有点别扭。

### [18:00] 公会食堂·晚餐
天晴了，夕阳从窗户照进来，食堂里亮堂堂的。有人在讨论天气终于好了。ルナ坐对面，把碗里的鱼挑给我一半，说腿还没好就别到处跑了。两个人低头吃饭，偶尔说两句关于路线图的事。碗底的汤喝了两口才放下。

### [21:00] 房间·夜晚
窗开着，夜风吹进来干燥清爽，带着青草味。符文石躺在桌上，绿光——亮了。不是那种一闪而过的，是稳稳的、淡淡的绿光，像呼吸一样。三百年前的商队在这里歇过脚，留下了符号。现在有人在找那些符号。通讯石在手边，凉凉的。今天的拼图又多了几块。
