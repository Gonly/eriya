# 主动消息记录

## 2026-07-02 Day49

### [05:00] 房间·醒来
天灰灰的，多云。没有阳光从窗帘缝里漏进来，空气闷闷的。昨晚翻来覆去睡不好，脑子里全是那个字——「瑟」？还是别的什么？符文石安静地躺在桌上，没有再亮。

### [07:00] 公会食堂·早餐
食堂里人不多，多云天大家似乎都起晚了。本叔在柜台后面打哈欠，看到我点了点头。嚼着面包的时候ルナ端着碗过来，说她昨晚翻到一本旧书，上面有个符号好像在哪里见过。

### [08:30] 塞拉斯研究室
塞拉斯把符文石传出的那个字拆成了两个音节。他说第一个音很轻，可能是「瑟」或者「瑟尔」，第二个音太弱了没录到。翻了四本旧档案，找到两个可能：一个是北方某地的地名开头，另一个是一个已经废弃的商队名称。

### [10:30] 石柱群东北侧·营地复查
走到昨天那堆灰烬旁边，蹲下来——脚印还在，但旁边多了一个符号，刻在一块扁平的石头上，三道斜线加一个圆圈。不是我刻的。正盯着看的时候身后传来脚步声。一个背着竹篓的老人从林子里走出来，竹篓里装着草药。他说他在这片林子采药好几年了，最近来过两三回。我问他有没有看到过别人，他说没有。检查了他的布袋——上面什么都没有。虚惊一场。但那个符号还在。

### [12:00] 公会食堂·午餐
把石头上符号的事告诉了本叔。他皱了皱眉说记下了。ルナ下午没来吃饭，说在查旧书。一个人嚼着面条，碗里的汤冒着热气，窗外天阴沉沉的。

### [14:00] 训练场·火魔法
多云天，空气里有水汽。掌心的火焰比昨天暗了不少。双段勉强过，第三段冲了一次——八秒就散了。手臂酸，心情有点烦。收招的时候火焰在指尖跳了两下就灭了。天气不好，没办法。

### [16:00] 房间·整理线索
把今天的发现写进笔记本：石头上的符号（三道斜线+圆圈）、采药老人的话、塞拉斯的音节分析。翻到之前画的营地地图，把符号标上去。ルナ下午过来了，翻开那本旧书指着一页——上面有一个几乎一模一样的符号。她说这本书是公会图书馆角落里翻出来的，讲的是某个北方商队的历史。不确定是不是同一个，但至少——有东西可以查了。

### [18:00] 公会食堂·晚餐
天暗得早，食堂里灯已经亮了。有人在讨论天气，说可能要下雨。ルナ坐对面，两个人低头吃饭，偶尔说两句关于符号的事。碗底的汤喝了两口才放下。

### [21:00] 房间·夜晚
窗半开着，风吹进来有点潮。符文石安静地躺在桌上，绿光没有再亮。通讯石在手边。今天的线索比昨天多了一点——符号、旧书、商队。拼图还缺很多块，但至少知道往哪个方向找了。
