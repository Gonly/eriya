## 灰袍商会（暂定名）

| 项目 | 内容 |
|------|------|
| **已知程度** | ⭐ 仅知存在 |
| **类型** | 商会的暗面（非法贸易） |
| **活动区域** | 跨城镇灰色地带 |
| **成员规模** | 中等（明面是正规商会，暗面人数不详） |
| **负责人** | 明面会长 × / 幕后 × |
| **与主线关系** | 🔗 间接关联 |

**概要**
表面上是一家普通商会，经营药材和矿石贸易。暗地里收购稀有魔法物品和古代遗物，不问来源。

**特点**
- 「灰」字表示在合法与非法的边界上游走
- 有实体店面，可以正常上门交易
- 但有些商品不摆在台面上——你得知道暗号或通过介绍人

**可能接触场景**
- 出售稀有采集物时被对方盯上
- エルマー表示「灰袍的人最近在打听发光矿石」
- 某个任务的目标物品被灰袍捷足先登

**与主线关联（浅层）**
灰袍商会是收集古代遗物的重要渠道之一。他们可能收走了一些先民遗物，但不知道真正的用途，只是以古董价格低买高卖。

## 追随者 / 狂信徒（暂定名）

| 项目 | 内容 |
|------|------|
| **已知程度** | ⭐ 仅知存在 |
| **类型** | 邪教 / 古代崇拜 |
| **活动区域** | 边境、偏僻村庄 |
| **成员规模** | 小规模 |
| **负责人** | 「先知」（身份未知） |
| **与主线关系** | 🔗 间接关联（后期可能引出主线反派） |

**概要**
一群崇拜先民魔法科技的人，认为大崩落不是灾难而是「神罚」，现在的世界是堕落的。他们试图复现先民的魔法科技来「净化世界」。

**特点**
- 非公开活动，在边境和偏远村庄秘密传教
- 成员以底层民众为主，被「先民的辉煌」吸引
- 部分成员有简单的魔法知识（从遗迹碎片中学来的）
- 组织松散，核心成员寥寥

**可能接触场景（C级后）**
- 偏远村庄的任务，发现村民行为古怪
- 任务目标被信徒抢先一步
- 在某个遗迹附近遭遇「他们在做奇怪仪式」

**与主线关联（深层）**
- 追随者的「先知」可能在无意中得到了部分真相
- 追随者的行动可能惊动了真正了解大崩落幕后的人（→ 引出主线反派）
- 第4-5章可能发现追随者和主线反派之间有关系
