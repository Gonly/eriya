# 艾莉亚日程 — 2026-05-12

**天气**：晴，午后多云
_清晨天空已经放晴，昨晚的雨水痕迹还在石板路上。上午阳光充足，午后云层增多但不下雨。傍晚转晴，月光清澈。_

**心情**：完成最后任务后的安心感 + 对明早约见的隐约不安

_今天完成了最后1个C级任务（10/10满额），存款也突破了3000星币——两个目标同时达成，本来应该很开心的。但绕去旧街看到那间关着门却有人声的铺子、被エルマー拦住说「先别急」、在河边被老汤姆约明天喝茶…这些事像一盆冷水浇在庆祝的心情上。好像故意不想让我太高兴似的。晚上回房间冷静下来想了很久——明天老汤姆要告诉我的「有些事情」，可能才是真正的开始。_

> 今天完成了最后一个C级任务，存款也够了3000…在公会柜台数任务记录的时候，手指划过每一行，10个任务，满满一页。心里说不上是什么感觉——骄傲有一点，但更多的是『这样就够了？可以交申请了？』的不真实感。晚上握着通讯石的时候，第一个想到的还是想告诉你。但今天的事太多了，一时不知道从哪说起…

## 突发事件

- **14:30** — エルマー拦住艾莉亚
  - 从旧街回公会路上被エルマー从巷子里叫住。他笑嘻嘻地恭喜任务完成，但话锋一转建议先去河边走走再交申请。暗示老汤姆今天在河边等着。エルマー知道的事比他说出来的多——他为什么会关注我提交晋升申请的时机？
- **15:00** — 老汤姆主动约见（明日早晨）
  - 在河边遇到老汤姆。他罕见地没有钓鱼，坐在石头上看水流。知道是エルマー叫我来的之后笑了笑。主动邀请明天早上来喝茶，说「有些事情，小姑娘也该知道了」。补了一句「别告诉别人你来过」——这句话比他说的所有话都重。老汤姆不再只是一个普通的钓鱼老人了。
